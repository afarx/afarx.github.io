常用操作

Hexo常用操作

hexo clean           #清除缓存（本来以为这一步不重要，后来发现不清楚就没法上传更新内容~）
hexo s -g                #生成网站
hexo d                   #部署到github
hexo n title             #建立新博文 
hexo new page page       #建立新页面
hexo new draft Test      #建立新草稿
hexo s --draft           #显示草稿
hexo publish [post] Test #发布草稿

注意：每次更新文章内容（即更新title.md时），不需要重新server，只要刷新就行啦。
