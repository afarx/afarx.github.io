连接Rstudio与Github

Author: Afar Date: February 26, 2018  23:27:45

这篇的版权属于勤奋的小伙伴 Zhimeng Xu
小伙伴脆骨~

假设大家对github基本操作已经熟悉，那么怎么将github和Rstudio整合呢，这样就可以直接在Rstudio更改完代码再push到远程仓库，或者直接在Rstudio中pull远程仓库的代码？

  • 打开Rstudio，右上角选择Project -> New Project...

  • 选择Version Control -> Git

    输入github仓库地址（如果这里不知道那你还需要再去熟悉一下github）和选择要保存在本地的路径就可以，将左下角勾选。点击Create Project就可以了。
  • 接下来就简单了，所看到的界面就有github基本的操作，如Commit，push (向上的绿色箭头)，创建分支，切换分支等。
  • 同样，Rstudio还提供命令行窗口，完全不必点鼠标。可到设置中先设置好命令行窗口的Shell。

    如果没有请更新到最新版Rstudio，这里我的Shell选择Git Bash无法使用尚未解决，所以我选择的是Command Prompt (64-bit)，也可以正常输入git命令行语句。
